# **App Name**: RadiusTalk

## Core Features:

- Anonymous Sign-in: Enable users to sign in anonymously for immediate access.
- Geo-Based Chatroom Creation: Allow users to create group chatrooms limited by a geographical radius.
- Real-Time Chat: Enable real-time messaging within the chatroom.
- Proximity Detection: Show users other active chatrooms based on their location within a specified radius using Firestore.
- Chatroom Listing: List nearby chatrooms, ordered by distance.
- Report User: Allow users to report inappropriate messages and content. Implement automated filtering based on user reports and generative AI-driven tool to detect toxic behavior patterns.

## Style Guidelines:

- Primary color: HSL (210, 70%, 50%) - A vibrant blue (#3399FF) to evoke a sense of connectivity and technology.
- Background color: HSL (210, 20%, 95%) - A light, desaturated blue (#F0F8FF) for a clean and calming background.
- Accent color: HSL (180, 60%, 40%) - A contrasting teal (#33BDBD) for interactive elements.
- Body and headline font: 'Inter' - A sans-serif font offering a modern and clean aesthetic for readability and a neutral tone.
- Use minimalist icons for chatroom locations, user profiles, and reporting features.
- Subtle animations for new messages and user entry/exit in chatrooms.
- Simple layout that clearly lists nearby chatrooms and their proximity to the user.
